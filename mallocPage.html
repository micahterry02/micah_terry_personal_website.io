Dynamic Memory Allocator

malloc, realloc, and free in C

In this project, I rewrote the core memory management functions in C: malloc, realloc, and free. The objective was to create a custom memory allocator that efficiently handles memory allocation and deallocation for all singly-threaded programs. This involved developing a memory management system capable of allocating, resizing, and freeing memory blocks effectively.

Key Skills and Concepts Learned:

  • Memory Management: Gained a deep understanding of memory management techniques, including allocation strategies, memory fragmentation, and garbage collection. This experience provided insights into how memory can be efficiently managed and reclaimed.
  • C Programming: Enhanced my proficiency in C programming, particularly in low-level programming and managing memory directly. This project required careful handling of pointers and memory addresses to ensure correct functionality.
  • Data Structures: Applied various data structures such as linked lists and block headers to manage free and allocated memory blocks. This involved designing and implementing efficient algorithms for memory allocation and deallocation.
  • Debugging: Developed advanced debugging skills to identify and resolve issues related to memory leaks, segmentation faults, and performance bottlenecks. This included using tools and techniques for thorough testing and validation.
  • Performance Optimization: Learned techniques for optimizing the performance of the memory allocator, including minimizing overhead and improving allocation speed. This experience taught me how to balance efficiency with functionality.

Overall, rewriting malloc, realloc, and free provided a profound understanding of memory management and enhanced my ability to write efficient, low-level code. The project reinforced my skills in C programming and memory optimization, laying a strong foundation for future system-level development.
